# Netflix Clone - Movie Infomation Application

<img width="1440" src='https://i.postimg.cc/WhzJfc4Y/image.png' border='0' alt='image'/>

Developed a multi-page movie information site with an engaging UI featuring animations, transitions, and carousels. Utilised React Router for seamless navigation across pages. Integrated REST APIs to fetch and display real-time movie data and implemented Stripe for secure payment functionality. Employed Git for version control and collaboration.

## Install Packages

```bash
npm i
```

## Start the app

```bash
npm run dev
```
